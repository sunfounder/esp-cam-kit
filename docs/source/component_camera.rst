カメラ
============

.. image:: img/camera.png
    :width: 200
    :align: center

OV2640カメラモジュールは、長さわずか21mmのコンパクトなサイズです。1/4インチのレンズとf/2.4の絞りを備えており、視野角は68度の広範囲をカバーします。このモジュールは200万画素の高解像度画像を提供し、最大1600x1200の解像度を誇ります。DVPインターフェースを使用しており、さまざまな用途に対応します。

**仕様**

* **モデル**: OV2640
* **長さ**: 21mm
* **画素数**: 200万
* **視野角**: 68°
* **インターフェース**: DVP
* **レンズサイズ**: 1/4インチ
* **解像度**: 1600x1200
* **絞り**: f/2.4
